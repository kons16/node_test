# node_test
expressを利用したsample

```
git clone https://github.com/kons16/node_test.git
cd node_test
node main.js
```

初期状態だと以下の URL の routing が設定されています
```
http://localhost:3000/             // routes/sample
http://localhost:3000/fruite       // routes/sample
http://localhost:3000/users/all    // routes/user
```
